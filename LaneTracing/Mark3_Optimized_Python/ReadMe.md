# Mark1 에서 코드최적화로 Upgrade

### 연산을 줄이기 위해 시도한 것
1. 불필요한 마스킹 연산 제거 (시각적으로 보이기 위함)
2. 카메라가 켜져있을 때 동작과 같은 if문 제거
3. 해상도 320x240 으로 낮춤
4. 사용하지 않는 cy 계산 제거
5. 제일 연산량 많이 잡아먹는 imshow 제거 → 추후 print문도 제거하면 좋음
6. if문 사용하여 만약 주행 명령이 바뀔 경우에만 딱 1번 명령내리도록 변경
